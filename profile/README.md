此專案是一個 Vtuber 聲音按鈕網站的實作，使用了資料分離式架構，使編輯內容只需撰寫 json 設定檔即可。  
前端使用 Angular，後端則採用 Azure Functions，音檔存放於 Azure Blob Storage。  
專案特色在於線上的音檔投稿系統，提交表單後能自動剪輯 Youtube 音訊並生成按鈕。
